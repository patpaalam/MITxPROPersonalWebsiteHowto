It's easy!

Here is how to create a Personal Website on Github!

  1. Go to Github.com and create an acount.


  2. After you create an acount go to your profile and select Repositories.

  3. Click "New" and select a name for your website on Github.
  4. At the bottom, press "Create Repository" button.
  5. Now that you have created a Repository, You have 2 options:
    • Create a New File on Github

      • Or
    • Upload an existing file on Github

  6. When you choose to Create a New File, you'll be able to type the code on Github.
  7. When you choose to Upload an Existing File it will prompt you to this page.

    You can drag and drop files here.
    Make sure you upload all the files involved in the website, including the Media Files.
  8. After you have Uploaded all files for your website, below you will need to Commit Changes
    Press Commit Change button
  9. And Lastly, in order for your newly created website to go live on Github make sure to:
    1. Go to Settings
    2. Go to Github Pages
    3. Under Sources select Branch to Main

  10. And there we go! Our Personal Website is Live and runing on Github.com

Thank you!
